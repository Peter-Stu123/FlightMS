数据库我已经创建了两个分别在后端工程中的Entity文件夹下，分别是Cityinfo和Flightinfo

在后端工程中，我已经创建了两个Controller文件夹下的CityinfoController和FlightinfoController
CityinfoController是城市信息表的控制器，FlightinfoController是航班信息表的控制器
